English

1. Support contact

  • Email: shteosis@gmail.com
  • Repository: github.com/Facta-Leopard/LoRAbit
  • Issues: GitHub Issues
  • Discussions: GitHub Discussions

Email is the primary direct support path for the current public build.

2. Support covers

  • app launch issues
  • workspace selection and file access issues
  • model inspection, preflight, dataset preparation, training, and export issues
  • questions about the current public build of the app itself

3. Support does not cover

  • granting rights to third-party models, datasets, or documents
  • changing licenses or usage terms from third-party providers
  • guaranteeing compatibility for every local model layout

4. Helpful details

When possible, include the compatibility report, preflight summary, training overview, run log, operating system version, device model, selected model input type, workspace type, and the exact warning or error shown on screen.
